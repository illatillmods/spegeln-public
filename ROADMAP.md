# Public Roadmap

This roadmap is public-safe and intentionally omits private strategy, source
details and implementation tactics.

## Current Phase

Private beta and release preparation.

Focus areas:

- legal review of publication, privacy, payments, batch workflows and API access
- reviewed screenshots and public demo material
- tighter release defaults for report publication and evidence handling
- clearer partner/onboarding process for API access
- operational review policies for civic modules

## Before A Public Launch

- finalize privacy and retention policy
- decide the public reporting/anonymity language after legal signoff
- verify deployment configuration and production secrets outside the mirror
- review all screenshots and examples for sensitive data
- confirm owner name and final rights notice
- decide whether a standardized source-available license is appropriate

## Not Planned For This Mirror

- source code publication
- prompt publication
- datasets or connector implementation details
- infrastructure configuration
- internal benchmarks until reproducible public-safe summaries exist
